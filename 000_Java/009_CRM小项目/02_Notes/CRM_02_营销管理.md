### 3.2 MySql模糊查询concat()函数

作用：连接字符串

用法：

```sql
select * from student where name like concat("%",#{name},"%");
```

:question: 原因：java开发中，查询条件一般是作为参数传递过来的，故需用concat()函数来拼接。
